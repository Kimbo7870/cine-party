# Cine Party
Movie night party games you can run in a browser, for family game nights

**Modes**
- **Guess the Movie (Poster / Tiles):** reveal the poster gradually and guess first
- **Guess the Trailer:** watch short clips and guess the film

## Demo
- Local: follow the quickstart below

## Tech Stack
- React + Vite
- JavaScript

## Quickstart (Local)
```bash
cd guess-movie
npm install
npm run dev
